# Stock Portfolio Tracker

A full-stack web application to track stock investments with live market prices, automatic gain/loss calculations, and portfolio analytics.

## Features
- User authentication (JWT-based, encrypted passwords)
- Live stock prices via Finnhub API
- Portfolio dashboard with performance charts and asset allocation
- Watchlist, transaction history, dividends tracking
- Price alerts, notes, and analytics
- Fully responsive design

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Recharts, React Router
- **Backend:** Node.js, Express
- **Database:** MongoDB (Mongoose)
- **Auth:** JWT, bcrypt
- **APIs:** Finnhub (stock prices), Yahoo Finance (indices)
